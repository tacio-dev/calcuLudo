
Uma calculadora simples feita com HTML, CSS e JavaScript puro.  
Projeto criado com o objetivo de praticar manipulação do DOM e lógica de programação.

## Funcionalidades
- Adição (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (÷)
- Botão de limpar (C)
- Botão de apagar (←)


## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
